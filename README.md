## 🎯 Problem Statement

Indian farmers struggle to find reliable information from government agricultural documents. General AI models like ChatGPT hallucinate, don't cite sources, and fail on specialized Indian agricultural knowledge.

## 💡 Solution

**KISAN MITRA AI** is a specialized AI platform that answers questions using ONLY uploaded government PDFs (ICAR manuals, PM-KISAN schemes, crop guides) with **exact page citations** - eliminating hallucinations completely.

## ✨ Key Features

| Feature | Description |
|---------|-------------|
| 🤖 **Krishi Sakha AI** | Farmer's trusted AI companion |
| 📄 **RAG Search** | Retrieval-Augmented Generation from uploaded PDFs |
| 🎯 **Exact Citations** | Every answer includes source + page number |
| 🌐 **10 Languages** | English, Hindi, Tamil, Telugu, Kannada, Malayalam, Marathi, Gujarati, Punjabi, Bengali |
| 🎤 **Voice Input** | Ask questions using speech recognition |
| 📊 **Agriculture Graph** | Interactive crop yield data visualization |
| 🏛 **Scheme Scanner** | Government scheme eligibility checker |
| 🌾 **Crop Advisor** | AI-powered crop planning and yield predictions |

## 🛠️ Technology Stack
Frontend: HTML5, CSS3, JavaScript (ES6)
AI Model: Groq Llama 3.3 (70B parameters)
RAG: PDF.js + Custom Retrieval
Charts: Chart.js
Animations: GSAP + AOS
Icons: Font Awesome 6
Voice: Web Speech API
